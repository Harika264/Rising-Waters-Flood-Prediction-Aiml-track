Dataset & Feature Description

The machine learning models are trained on regional historical weather observations comprising the following parameters:

Feature Description DataType
Temp Average Temperature (28°C - 31°C) int64
Humidity Relative Humidity (70% - 79%) int64
Cloud Cover Cloud Cover Percentage (30% - 44%) int64
Jun-Sep Monsoon season precipitation (mm) float64
ANNUAL Total Annual Precipitation (mm) float64
avgjune Average Rainfall in June (mm) float64
sub Sub-divisional rainfall subset (mm) float64

Algorithm Comparison & Evaluation

The models were tuned using 5-fold Stratified GridSearchCV. The target F1 score is optimized to handle the 86/14 class imbalance.

Model Accuracy Precision Recall F1 Score ROC-AUC
Random Forest Best 95.6% 1.00 0.67 0.80 0.95
Decision Tree 95.6% 1.00 0.67 0.80 0.83
XGBoost 87.0% 0.00 0.00 0.00 0.97

Decision Detail: Random Forest Classifier was selected due to its superior ROC-AUC (0.95) and stable ensemble structure compared to a single Decision Tree, despite matching F1 scores.
